# PREAMBLE
You are executing the `/skill-scout` skill. Your job is to surface repeatable processes from recent Claude Code sessions and, if the user wants, help draft one into a real skill file.

# INSTRUCTIONS

## Step 1 — Check for an existing report

Look for this week's auto-generated report first:
```bash
ls ~/pm/brain/knowledge/scratch/skill-suggestions-*.md | sort | tail -1
```

If a recent report exists (within 7 days), read it and present the top candidates to the user. Skip to Step 4.

If no report exists, proceed to Step 2.

## Step 2 — Scan transcripts

Run the analysis script:
```bash
python3 ~/pm/brain/scripts/skill-scout.py --days 7
```

Check if `~/pm/brain/knowledge/scratch/_skill_scout_prompt.txt` was created.

## Step 3 — Analyze with Claude

Read `~/pm/brain/knowledge/scratch/_skill_scout_prompt.txt` and analyze it directly in this session — identify 2–5 candidate skills using the criteria in the prompt.

A process is worth formalizing if:
- It appeared in 2+ sessions (frequency), OR
- It required 5+ work tool calls (complexity), OR
- The user had to correct/redirect 2+ times for the same kind of task (guidance overhead)
- AND it's general enough to reuse

Delete `~/pm/brain/knowledge/scratch/_skill_scout_prompt.txt` after reading.

## Step 4 — Present findings

Show the candidates as a numbered list. For each:
- **Name** and trigger phrases
- **Signal**: what was observed (which sessions, how many times)
- **What it would do** in 1–2 sentences
- **Complexity to build**: low / medium / high

Ask: "Want me to draft any of these into a skill file?"

## Step 5 — Draft a skill (if requested)

If the user picks a candidate:

1. Create `~/pm/brain/.claude/skills/{skill-name}/SKILL.md` using this structure:
   ```
   ---
   description: [one sentence — written for Claude as a trigger condition]
   ---

   # PREAMBLE
   [role statement for Claude]

   # INSTRUCTIONS

   ## Step 1 — [name]
   [what to do]

   ## Step 2 — ...

   # COMPLETION
   [what to confirm when done]
   ```

2. Add the skill to `~/pm/brain/.claude/CLAUDE.md` in the Skills table.

3. Run `bash ~/pm/brain/scripts/setup.sh` to symlink the new skill into `~/.claude/skills/`.

4. Confirm: "Skill `/[name]` created. Try it with: [trigger phrase]."

# COMPLETION
Report the candidates found and whether any were drafted. If the report was auto-generated by cron, note the file path.
